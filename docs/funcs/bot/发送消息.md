# 发送消息(send_msg)
## 使用例子
```python
import AfdianBot

bot = AfdianBot.Bot(account="xxx",password="xxxx") # 实例化一个bot

@bot.register("/test1")
def test1():
    bot.send_msg("Hello1!")

bot.run()
```
如上，该函数可以自动地将指定消息发送至对方
## bot.send_msg 函数解析

> 发送消息,只支持文本消息  
:param user_id: 用户id  
:param msg: 消息内容

| 字段名     | 类型     | 描述      |
|---------|--------|---------|
| msg     | string | 消息内容    |
| user_id | string | 目标用户 ID |
