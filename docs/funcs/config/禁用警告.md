# 禁用警告(disable_warnings)
## 使用实例
```python
import logging
import AfdianBot

logger = logging.getLogger("AfdianBot")
AfdianBot.config.disable_warnings()
logger.warning("这是一条警告") # 此时不会再输出了
```

## config.disable_warnings 函数解析

### 输入参数
| 字段名    | 类型      | 描述       |
|--------|---------|----------|
| status | boolean | 状态(是否禁用) |
