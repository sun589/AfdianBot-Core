# 获取赞助方案(get_plans)
## 使用实例
```python
import AfdianBot

print(AfdianBot.api.get_plans(user_id="xxx"))
```

## api.get_plans 函数解析

> 查询指定用户的赞助方案

### 输入参数
| 字段名        | 类型     | 描述     |
|------------|--------|--------|
| user_id    | string | 目标的用户id |

### 返回数据
List类型

```json
[
    {
        "plan_id":"xxxx",
        "name":"xxxx",
        "price":11.45,
        "desc":"example1"
    }, 
    {
        "plan_id":"xxxx",
        "name":"xxxx",
        "price":23.33,
        "desc":"example2"
    }
]
```
| 字段名     | 类型     | 描述   |
|---------|--------|------|
| plan_id | string | 方案id |
| name    | string | 方案名称 |
| price   | float  | 方案金额 |
| desc    | string | 方案介绍 |
