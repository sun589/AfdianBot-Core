# 登录(login)
## 使用实例
```python
import AfdianBot

print(AfdianBot.api.login(account="xxx",password="xxx")) # 登录
```

## bot.register 函数解析

> 登录爱发电，返回auth_token

### 输入参数
| 字段名      | 类型     | 描述 |
|----------|--------|----|
| account  | string | 账号 |
| password | string | 密码 |

### 返回参数
爱发电登录返回的auth_token
