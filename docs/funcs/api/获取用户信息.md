# 获取用户信息(get_user_info)
## 使用实例
```python
import AfdianBot

print(AfdianBot.api.login(user_id="xxx"))
```

## bot.register 函数解析

> 查询指定用户的信息

### 输入参数
| 字段名      | 类型     | 描述 |
|----------|--------|----|
| account  | string | 账号 |
| password | string | 密码 |

### 返回参数
Dict类型

| 字段名    | 类型     | 描述    |
|--------|--------|-------|
| name   | string | 用户名   |
| avatar | string | 头像url |

如果想查询用户赞助的金额，详见[查询赞助者信息(query_sponsor)](/docs/funcs/api/查询赞助者信息.md)
