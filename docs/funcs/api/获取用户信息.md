# 获取用户信息(get_user_info)
## 使用实例
```python
import AfdianBot

print(AfdianBot.api.get_user_info(user_id="xxx"))
```

## api.get_user_info 函数解析

> 查询指定用户的信息

### 输入参数
| 字段名     | 类型     | 描述         |
|---------|--------|------------|
| user_id | string | 用户的user_id |

### 返回数据
Dict类型

| 字段名    | 类型     | 描述    |
|--------|--------|-------|
| name   | string | 用户名   |
| avatar | string | 头像url |

如果想查询用户赞助的金额，详见[查询赞助者信息(query_sponsor)](/docs/funcs/api/查询赞助者信息.md)
